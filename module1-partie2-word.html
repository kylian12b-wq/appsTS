3. Styles de titres et sommaire automatique

📋 Hiérarchie des styles

Titre 1 — Chapitre principal (ex: 1. Présentation du projet)
  Titre 2 — Sous-partie (ex: 1.1 Description des travaux)
    Titre 3 — Sous-sous-partie (ex: 1.1.1 Menuiseries intérieures)
      Corps de texte normal — votre contenu rédigé ici...

🔖 Créer un sommaire automatique en 3 étapes

1
Applique les styles Titre 1, Titre 2, Titre 3 à tes titres
2
Clique à l'endroit où tu veux le sommaire
3
Références → Table des matières → choisir un style → OK

Mise à jour : Clic droit sur le sommaire → Mettre à jour les champs → Mettre à jour toute la table

📌 Fiche Récap — Word Pro

✅ Checklist avant d'envoyer un document Word

  • Orthographe vérifiée (F7 ou Révision → Orthographe)
  • Mise en page cohérente (marges, police, interligne)
  • Styles de titres appliqués correctement
  • Sommaire mis à jour (clic droit → Mettre à jour)
  • En-tête et pied de page présents
  • Numéros de page corrects
  • Tableaux bien alignés et lisibles
  • Sauvegarde en .docx ET en .pdf si nécessaire

❌ Erreurs classiques à éviter

  • Faire les titres manuellement (gras + grande police) au lieu des styles
  • Oublier de mettre à jour le sommaire avant d'envoyer
  • Mélanger plusieurs polices sans cohérence
  • Laisser des sauts de page manuels (Entrée x10) au lieu d'un vrai saut
  • Envoyer en .docx quand le client n'a pas Word (utiliser .pdf)

🛠️ Exercices Pratiques

Exercice 1 — Mettre en page un document professionnel

Objectif : Créer un document Word correctement mis en page pour une entreprise de menuiserie.

Consignes :

  1. Créer un nouveau document Word
  2. Régler les marges : 2.5cm partout, format A4, orientation Portrait
  3. Choisir la police Calibri 11 pour le corps de texte
  4. Ajouter un en-tête avec : Nom entreprise | Titre document | Date
  5. Ajouter un pied de page avec : Confidentiel | Page X/Y | Votre nom
  6. Rédiger une page de contenu avec 3 titres de niveau différent

✅ Corrigé

Mise en page → Marges → Marges personnalisées → 2.5cm. Police : Accueil → sélectionner Calibri → taille 11. En-tête : Insertion → En-tête → Vierge. Pour la date : Insertion → Date et heure. Pied de page : Insertion → Pied de page → Numéro de page au centre.

Exercice 2 — Compte-rendu de réunion de chantier

Situation : Tu es conducteur de travaux sur un chantier cuisine. Rédige le compte-rendu de la réunion du 15/04/2026.

Participants : Client M. Dupont, Architecte Mme Leblanc, Menuisier M. Martin (toi)

Points abordés : Retard livraison façades MDF, modification couleur plan de travail, validation pose électroménager

✅ Structure attendue

En-tête avec date/lieu/participants → Avancement (cuisiniste : 60%, électricité : 100%) → Point bloquant : façades repoussées au 22/04 → Décision : client valide gris anthracite pour plan de travail → Prochaine réunion le 22/04.

Exercice 3 — Rapport technique avec sommaire automatique

Objectif : Créer un rapport technique de 4 pages avec sommaire automatique fonctionnel.

  1. Créer les chapitres : Introduction / Présentation du chantier / Choix techniques / Conclusion
  2. Appliquer le style Titre 1 à chaque chapitre
  3. Ajouter 2 sous-parties par chapitre avec le style Titre 2
  4. Insérer un sommaire automatique page 2
  5. Ajouter un tableau matériaux avec au moins 5 lignes

✅ Points clés du corrigé

Styles : sélectionner le texte → Accueil → cliquer sur "Titre 1" dans le panneau Styles. Sommaire : se placer page 2 → Références → Table des matières → Table automatique 1. Mise à jour finale : clic droit → Mettre à jour les champs → Mettre à jour toute la table.
